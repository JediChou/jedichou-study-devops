Shell In Action Study

### Shell
* Linux Shell脚本攻略, Sarath Lakshman, 门佳(译)
* Linux Shell脚本攻略（第2版）, Shantanu/Sarath Lakshman, 门佳(译)

### PowerShell
* Windows PowerShell in Action, Bruce Payette
* Windows PowerShell应用手册, Lee Holmes, 赵松德/王英群(译)
* Powershell and WMI, Richard Siddaway
* Windows PowerShell Cookbook, 
* The Personal Trainer Windows PowerShell for Administration, William Stanek
